# Markdown in HTML tag

> Good Markdown!

Markdown content can be wrapped in HTML tags to achieve some effect.  

```markdown
<div style='color: red'>

- listitem
- listitem
- listitem

</div>
```

You can also render Markdown content in the details element.  

```markdown
<details>
<summary>Self-assessment</summary>

- Great
- Beautiful

</details>
```
